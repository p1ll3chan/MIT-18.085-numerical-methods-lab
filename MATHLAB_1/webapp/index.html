About This Repository

The content originates from coursework (18.085) focusing on computational science and engineering. It includes implementations in both C++ and Python demonstrating numerical methods for solving systems of equations that arise in physical modeling, particularly mass-spring systems and boundary value problems.

The Thomas Algorithm implementation showcases an optimized approach to Gaussian elimination for tridiagonal matrices, which commonly appear when discretizing differential equations using finite difference methods.
